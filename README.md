# Flowboard

Flowboard est un tableau Kanban interactif développé en React et TypeScript. Ce projet a été réalisé pour mettre en avant mes compétences en développement front-end.

## Objectif

Ce projet sert de vitrine pour démontrer mes compétences dans les technologies suivantes :

- **React** — Architecture en composants, gestion de l'état avec le pattern reducer (`useReducer`)
- **TypeScript** — Typage strict, interfaces et types personnalisés
- **CSS / HTML** — Mise en page responsive avec CSS Modules
- **Tests unitaires** — Couverture de tests avec Vitest et React Testing Library

## Fonctionnalités

- Tableau Kanban avec trois colonnes : *Ideas*, *Doing*, *Done*
- Ajout et suppression de notes
- Support de l'internationalisation (i18n) — anglais et français

## Stack technique

- [React](https://react.dev/) + [TypeScript](https://www.typescriptlang.org/)
- [Vite](https://vitejs.dev/) pour le bundling et le serveur de développement
- [Vitest](https://vitest.dev/) + [React Testing Library](https://testing-library.com/) pour les tests
- CSS Modules pour le styling

## Lancer le projet

```bash
npm install
npm run dev
```

## Lancer les tests

```bash
npm test
```
